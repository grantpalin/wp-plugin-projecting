Provides the means to create and organize projects being worked on currently or in the past.

== Description ==

Provides a custom post type named Project, managed similarly to pages. Currently supports standard title and content fields.

== Installation ==

= Manual Installation =

1. Upload the entire `/projecting` directory to the `/wp-content/plugins/` directory.
2. Activate Projecting through the 'Plugins' menu in WordPress.
